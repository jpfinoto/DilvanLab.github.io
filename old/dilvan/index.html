Courses

     
    Sistemas Operacionais - Operating Systems - A graduate course I teach at USP (since August 1998). It covers Unix, Distributed Programming and Parallel Machines.

    Seminários Avançados em Hipermídia - Advanced Seminars in Hypermedia- A graduate course I teach at USP (since March 2003). Seminars covering hypermedia technologies.
     
    Programação Orientada a Objetos - Object Oriented Programming - An  undergraduate courses I teach at USP (since May 1998). It covers object oriented programming and the Java language.
    Sistemas Operacionais II - Operating Systems II - An undergraduate course I teach at USP (since May 1997). 
    Microcomputadores e Microprocessadores - Microcomputers and Microprocessors - An undergraduate course I teach at USP (since March 1999). It is composed only by laboratory classes about computer hardware.
