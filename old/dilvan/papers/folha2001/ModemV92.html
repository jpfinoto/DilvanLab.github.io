TESTE USP

Avaliação revela vantagens, mas indica que compra deve ser pensada

Modem V.92 só vale se provedor adotar tecnologia

DILVAN DE ABREU MOREIRA
ESPECIAL PARA A FOLHA

O Modem de 56 Kbps modelo 5686D, da U.S.Robotics, apresenta características semelhantes às de outros modelos de modems externos do mesmo fabricante, com a exceção de que ele permite conexões utilizando tanto a tecnologia V.90 quanto a nova tecnologia V.92. O modem vem com um CD de instalação e um CD com softwares. A tecnologia V.92 inclui três novos recursos.
O Modem on Hold permite que o usuário suspenda uma conexão de dados enquanto atende uma ligação telefônica, sem perder a conexão com a rede, o que possibilita uma melhor utilização de uma única linha telefônica. Essa função requer que o serviço de chamada em espera da operadora local esteja habilitado.
O Quick Connect permite que o modem compare uma nova conexão com uma feita anteriormente, verificando se as condições da linha são semelhantes. Se forem, o modem deixará de executar alguns passos do processo de conexão, tornando-a mais rápida.
O V.PCM Upstream permite alcançar velocidades de upload de até 48 Kbps (a tecnologia V.90 limita essa velocidade a 31,2 Kbps). O usuário pode balancear as taxas de download e upload, pois, ao aumentar uma das taxas, a outra é reduzida. Ou seja, o usuário não conseguirá fazer downloads a 56 Kbps enquanto estiver fazendo uploads a 48 Kbps.
Para usar esses novos recursos da tecnologia V.92, é necessário que o provedor de acesso à internet suporte conexões com ela.
Os provedores de internet não atualizaram seus servidores para suportar a tecnologia V.92 (a maioria usa a V.90), o que inviabiliza, em princípio, o aproveitamento desses novos recursos.
O modem foi testado em conexões com provedores de internet que suportam a tecnologia V.90.
Nesse teste, o modem desempenhou funções e se comportou como os modelos de 56 Kbps.
Em um segundo teste, foram utilizados dois computadores, ambos equipados com o U.S.Robotics 5686D. Nesse caso, como os dois modems têm capacidade de comunicação com a tecnologia V.92, pôde-se perceber uma das características do novo padrão: a conexão rápida (QuickConnect). O modem gastou menos tempo para se conectar após a realização de conexões sucessivas.

Instalação
Para instalar o modem, o usuário deve verificar se seu computador possui uma porta serial livre e habilitada. Deve-se dar preferência à porta COM2, onde o modem pode ser instalado de forma mais eficiente (se instalado na porta COM1, talvez não seja reconhecido corretamente).
A conexão do modem com o computador deve ser feita por um cabo serial do padrão RS232 (não vem com o modem).
A instalação do modem é relativamente simples se comparada à de um modem interno, desde que a porta serial do computador esteja corretamente configurada. No entanto o manual contém algumas informações incorretas, que podem confundir até mesmo os usuários mais experientes.
Uma das formas mais simples de instalar o modem é fazê-lo com o micro e o modem desligados.
Conecte os cabos como indicado no manual de instalação.
Ligue o modem e, em seguida, o computador; quando o Windows pedir a localização do driver, coloque o CD de instalação e forneça o diretório.
Para usar o modem, é preciso estabelecer uma conexão dial-up com um provedor de internet. Para configurá-lo, é necessário instalar o software ControlCenter.
Esse programa, que acompanha o CD de instalação, configura as funções do modem e o controle sobre as taxas de comunicação descritas na tecnologia V.92.
No entanto o software que está no CD não funcionou corretamente com o modem. Foi necessário fazer o download de uma versão atualizada no site da U.S.Robotics, em www.usr.com/v92/v-download.asp.
Para configurar o modem e aproveitar os novos recursos oferecidos, use o manual do software ControlCenter, que explica detalhadamente todas as opções de configuração. Porém, apesar de o manual instruir o usuário a atualizar o software do modem depois da instalação, isso não deve ser feito, pois o modelo 5686D já vem atualizado para a tecnologia V.92.
O modem testado se destina a usuários domésticos. Deve-se pensar bem antes de adquiri-lo, pois não há um ganho significativo de velocidade. Ele é caro e o uso dos novos recursos, oferecidos pela tecnologia V.92, somente será possível se os provedores passarem a adotá-la.
É provável que a tecnologia V.92 nem chegue a ser usada pelos provedores, uma vez que tecnologias como ADSL e modem a cabo têm-se tornado cada vez mais atraentes, por terem melhor desempenho e pelo fato de os preços estarem cada vez mais baixos.
Embora a quantidade de internautas com conexão discada ainda seja muito grande, espera-se que ela diminua consideravelmente até 2004.

Dilvan Moreira é Ph.D em engenharia eletrônica pela University of Kent at Canterbury, Inglaterra. Colaboraram Elaine Silva e Richard Lingner.
