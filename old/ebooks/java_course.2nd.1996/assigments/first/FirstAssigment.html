First Homework Assigment (due 23 September):

    This assigment is to be done by groups of up to three students. Each group has to do one of the programs bellow. Please send me an e-mail with the name of the students in the group and the program you want to do:

    String changer

      This program should read a text file, search for a string and substitute this string for another. The input of commands for this program is done at the command line:

      changestr [-h] "string_to_change" "new_string" <file_name>

      The search and substitute algorithms have to be as eficient as you can make them. The -h option prints a small help about thr program.

    Lisp interpreter

      This program should must be able to read a text file, parse and execute the following lisp commands:

        (CAR <list>)            X= <string> or <list>
(CDR <list>)
(PRINTLN X)
(LIST X {X} )
        For instance, the file may have:
          (PRINTLN (CAR (CDR (LIST AB CD EF))))
(PRINTLN (CDR (CAR (LIST (LIST HY HJ I) HJ))))
        Output:
          CD
(HJ I)

    Uuencoder

      This program should read a file and encode it in less than 8 bits. It should rewrite the file using just the lower part of a byte. The command input for it is:

      uuencoder [-n <number_of_bits_used>] <file_name> <file_name_output>

      The optional argument -n will give the number of bits used to encode the file. The default value (and maximum) is 7 the minimum is 4.

    Interval number class

      A new class, derived from Number, that operates with intervals. An interval is a real number that has a precision range associated to it. For instance, the value of resistence of comercial resistors is given by an interval number. A 20K 10% resistor has its value in between 18K and 22K.

      This new class has to be able to perform the four basic arithmetic operations: adtion, subtraction, multiplication and division. It should have, at least, the following:

      class Interval extends number{
   Interval(float number, float precision);
   Interval(double number, float precision);
   float precision();
   demo();
   Interval add(Interval x);
   static Interval add(Interval x, Interval y);
      (The same for sub, mult, div)
      (Implement all Number methods)
      (Implement the possible Object methods)
}
      For 20K 10%: Interval(<number>, <precision%>) eg: Interval(20000, 10)

      For instance, if one wants to calculate the resistence of two parallel resistors:

      R1= 20K 10%   R2= 10K 5%
        Interval r1= new Interval(20000, 10);
Interval r2= new Interval(10000, 5);
Interval rp;
        rp= Interval.div( Interval.mult(r1, r2), Interval.add(r1, r2));
System.out.println(" Parallel: " + r1 + " " + r2 + " = " + rp);
      Output:
Parallel: 20000 10% 10000 5% = 6662.9 6.7%

      This class should have a demo() method to print out and explain its main capabilities. You may add to the class any functionality you think is worth.
