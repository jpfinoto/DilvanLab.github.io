Third Homework Assigment (due 18 November):

    This assigment is to be done by groups of up to three students. Each group has to do one of the programs bellow. Please send me an e-mail with the program you want to do:

    FTP viewer

      This program should implement a graphic front for a FTP client, that is able to provide the same functionality the following commands offer:

      • ls - List the current directory in the server.
      • cd - Go to a directory in the server.
      • lls - List the current directory in the client.
      • lcd - Go to a directory in the client.
      • put - Put a file in the server current directory.
      • mput - Put all files that match a wildcard in the server current directory.
      • get - Get a file from the server current directory.
      • mget - Get all files that match a wildcard from the server current directory.

      It just have to work with anonymous ftp and with only bin mode.

      The main Window should have a Help button explaining the functions and operations. It has to have an area to show the directory tree of the server machine. The user should be able to chose files from this area and download them to his machine, and to upload files to the directory represented in this area from a Dialog window. The main window should have a button to open ftp locations.

    WWW viewer

      This program should display HTML pages as links. The user will be able to point the tool to a HTML page and the tool will show all links in that page. The user then can click in one of those links and the tool will:

      • Show the new HTML page's links, if the link is a HTML page (Its name finishes in .html or .htm).
      • Show the figure in another window, if the link points to a .gif or .jpg file.
      • Show a message saying it cannot display that link, if any other kind of link (like Java applets).

      The program should have a HELP button that will display a little help and an OPEN button to open URLs.
