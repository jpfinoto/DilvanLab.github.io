The Java Language: Operators






    Bitwise Operators

    Operator     Use          Operation
   >>     op1 >> op2      shift bits of op1 right by distance op2
   <<     op1 << op2      shift bits of op1 left by distance op2
   >>>    op1 >>> op2     shift bits of op1 right by distance op2 (unsigned)
   &      op1 & op2       bitwise and
   |      op1 | op2       bitwise or
   ^      op1 ^ op2       bitwise xor
   ~      ~ op            bitwise complement







    Assigment Operator

    You use the assignment operator, =, to assign one value to another.



    Short Cuts for the Assigment operator:

    Operator     Use          Equivalent to
   +=      op1 += op2     op1 = op1 + op2
   -=      op1 -= op2     op1 = op1 - op2
   *=      op1 *= op2     op1 = op1 * op2
   /=      op1 /= op2     op1 = op1 / op2
   %=      op1 %= op2     op1 = op1 % op2

   &=      op1 &= op2     op1 = op1 & op2
   |=      op1 |= op2     op1 = op1 | op2
   ^=      op1 ^= op2     op1 = op1 ^ op2
   <<=     op1 <<= op2    op1 = op1 << op2
   >>=     op1 >>= op2    op1 = op1 >> op2
   >>>=    op1 >>>= op2   op1 = op1 >>> op2
