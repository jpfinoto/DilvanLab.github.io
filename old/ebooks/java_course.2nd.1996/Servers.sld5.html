Distributed Programming: URLs




URLs


What Is a URL?

Definition:

URL is an acronym that stands for Uniform Resource Locator and is a reference (an address) to a resource on the Internet.

Components:

  • Protocol identifier:
    • http://www.unicamp.br:80/lecture.html#TITLE
    • ftp://ftp.fee.unicamp.br/linux/Read.me
    • jdbc:msql://lcaee.fee.unicamp.br:4333/group1
  • Resource Name:
    • http://www.unicamp.br:80/lecture.html#TITLE
    • ftp://ftp.fee.unicamp.br/linux/Read.me
    • jdbc:msql://lcaee.fee.unicamp.br:4333/group1

Resource name format (protocol dependent):

host name
www.unicamp.br
the name of the machine the resource lives on.
filename
/lecture.html
the pathname to the file on the machine.
port number
:80
the port number to connect to (this is typically optional).
reference
#TITLE
a reference to a named anchor within a resource.
