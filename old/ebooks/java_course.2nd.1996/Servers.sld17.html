    if (!accessOK())

Installing Your Security Manager

      try {
              System.setSecurityManager(new PasswordSecurityManager("Booga Booga"));
          } catch (SecurityException se) {
              System.out.println("SecurityManager already set!");
          }

Note: The security manager for an application can be set only once. Typically, a browser sets its security manager during its startup procedure.
