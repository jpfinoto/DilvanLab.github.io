GUI: Layout Managers

GridLayout



Behavior:

GridLayout places components in a grid of cells. Each component takes all the available space within its cell, and each cell is exactly the same size. If the GridLayout window is risized, the GridLayout changes the cell size so that the cells are as large as possible.


Code Example:

          //Construct a GridLayout with 2 columns and an unspecified number of rows.
          setLayout(new GridLayout(0,2));
          setFont(new Font("Helvetica", Font.PLAIN, 14));
             
          add(new Button("Button 1"));
          add(new Button("2"));
          add(new Button("Button 3"));
          add(new Button("Long-Named Button 4"));
          add(new Button("Button 5"));


GridLayout constructors:

          public GridLayout(int rows, int columns)
          public GridLayout(int rows, int columns,
                            int horizontalGap, int verticalGap)
