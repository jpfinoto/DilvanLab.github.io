The getProperties() returns a Properties object.

The setProperties() method: takes a Properties object.

Forcing Finalization and Garbage Collection

System.runFinalization(): This method calls the finalize() methods on all objects that are waiting to be garbage collected.

System.gc(): It asks the garbage collector to run at any time by calling System's gc() method.

The Runtime Object

It bypasses the system-independent interface of the System class and use system resources directly from the runtime environment.

The oval labelled Runtime in the diagram represents the current runtime environment and is an instance of the Runtime Class.

Note: Messaging the Runtime object directly compromises your ability to run your program on different systems. You should do this only in special situation
