A API

A API objeto de estudo do presente trabalho é composta por três pacotes que juntos constituem o SDK Java Technology. Temos:

  • pacote javax.speech: possui as interfaces e implementações da classe Engine e implementa a classe Central para acesso a todas as outras funcionalidades. É a base para os demais pacotes, sendo que o reconhecimento e a síntese, como já visto, são herdeiros da classe Engine deste pacote. Possui também funcionalidades para gerenciamento do áudio recebido pelo Engine, além da classe SpeechEvent herdada por todos os outros eventos da API.
  • pacote javax.speech.synthesis: possui todas as classes e interfaces utilizadas para a realização da síntese, possui também a classe Voice que determina como será a síntese propriamente dita.
  • pacote javax.speech.recognition: o maior dos pacotes, implementa as classes de reconhecimento, as classes de gramáticas dada a relação entre os reconhecedores e estas entidades e as classes de resultados.

Os pacotes estão presentes em um arquivo jar denominado ibmjs.jar e a compilação das aplicações só é possível adicionando-se o caminho onde se encontra este arquivo à variável de sistema CLASSPATH. Isto pode ser feito da seguinte forma:

CLASSPATH = %CLASSPATH% ; .../ibmjs.jar
