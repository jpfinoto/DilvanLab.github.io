4.1. XSL transformation

Create two XSLT sheets to convert to HTML the files from the XML language you created as an exercise in the last chapter. Each XSLT sheet should take all the information in XML and display it in different formats (change background, presentation order, fonts, etc.).

To generate and test the XSLT files you may use the MS IExplorer, the Xalan - XSL stylesheet processors (http://xml.apache.org/#xalan), or any other tool you like.
