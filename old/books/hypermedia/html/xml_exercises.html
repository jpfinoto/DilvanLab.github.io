5. Exercises

5.1. XML generation

Create a small language derived from XML to describe a small dataset (i.e. contacts information, racing cars, kinds of animals, etc). This language has to have at least 10 tags and 5 attributes. It should be described using a DTD file. Write at least 3 files using this language.

To generate and test the DTD and XML files, you can use any XML parsing tools. As a suggestion you can use the XMLMind editor http://www.xmlmind.com (a free tool) or the XMLSpy editor http://www.xmlspy.com (free demo).
