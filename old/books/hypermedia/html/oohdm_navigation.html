1. OOHDM Navigation

The Object-Oriented Hypermedia Design Method (OOHDM) uses abstraction and composition mechanisms in an object oriented framework to, on one hand, allow a concise description of complex information items, and on the other hand, allow the specification of complex navigation patterns and interface transformations.

There is a link to the OOHDM site: http://www.telemidia.puc-rio.br/oohdm/oohdm.html

This chapter focus only on the navigational aspects of OOHDM, as a complementation for the UML models:

  • Introduction to OOHDM.

  • Navigational Project.

  • Case study: Online Publication.

As an exercise, you can draw parallels and differences between the UML diagrams and the OOHDM diagrams. You can use the complete OOHDM slides and the OOHDM specs.
