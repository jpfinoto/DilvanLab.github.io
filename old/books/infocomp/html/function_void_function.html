3. void Functions

The void function provide a way of emulating Pascal type procedures, creating functions that return nothing.

If you do not want to return a value you must use the return type void and miss out the return statement:

void squares()
{
   int loop;

   for (loop=1; loop<10; loop++)
      printf("%d\n",loop*loop);
}
 
main() 
{
   squares();
}
Important

We must have () even for no parameters unlike some languages.
