3. Looping

This chapter will look at C's mechanisms for controlling looping and iteration. These mechanisms look familiar and indeed will operate in standard fashion most of the time.

3.1. The for statement

The C for statement has the following form:

for  (expression1; expression2; expression3)
   statement;
   or {block of statements}

expression1 initialises; expression2 is the terminate test; expression3 is the modifier (which may be more than just simple increment);

NOTE: C basically treats for statements as while type loops

For example:

int x;

main() {
   for (x=3; x>0; x--) {
      printf("x=%d\n",x);
   }
}

...outputs:

x=3
x=2
x=1

...to the screen

3.2. The while statement

The C while statement has the following form:

while (expression)
   statement

It will execute the statement while expression is true. For example:

int x=3;

main() {
   while (x>0) {
      printf("x=%d\n",x);
      x--;
   }
}

...outputs:

  x=3
  x=2
  x=1

...to the screen.

3.3. The do-while statement

The C do-while statement has the following form:

do
   statement
while (expression)

It will execute the statement while expression is true, statement will be always executed at least once. For example:

int x=3;

main() {
   do {
      printf("x=%d\n",x);
      x--;
   } while (x>0)
}

...outputs:

  x=3
  x=2
  x=1

...to the screen.

3.4. break and continue statements

C provides two commands to control how we loop:

  • break -- exit form loop or switch.

  • continue -- skip 1 iteration of loop.

Consider the following example where we read in integer values and process them according to the following conditions. If the value we have read is negative, we wish to print an error message and abandon the loop. If the value read is great than 100, we wish to ignore it and continue to the next value in the data. If the value is zero, we wish to terminate the loop.

int value;

main() {
   scanf("%d", &value);
   while (value != 0) { 

      if (value < 0) {
         printf("Illegal value\n");
         break;
         /* Abandon the loop */
      }

      if (value > 100) {
         printf("Invalid value\n");
         scanf("%d", &value)
         continue;
         /* Skip to start loop again */
      }

      /* Process the value read */
      /* guaranteed between 1 and 100 */
      ....;

      scanf("%d", &value);
   } /* end while if value <= 0 */
}
