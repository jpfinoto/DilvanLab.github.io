This exercise is going to focus in solving a mathematical problem using C. A problem will be proposed, you should:

  1. Write an algorithm to solve the proposed program.

  2. Write a C program, following the algorithm.

  3. Test the program to show that it can solve the proposed problem.

The problem is: To calculate the exponential function using the Maclaurin series.

Computers CPUs (Central Process Units) are only capable of executing the four operation (+,-,*,/) as machine language instructions. How they can calculate mathematic functions, like sine, cosine, etc? They do it in the same way us humans do: Using mathematical series (such as Taylor, Maclaurin, etc).

The proposed problem is then to write a program, using the Maclaurin series for exponential, that can calculate the exponential function for float numbers.
