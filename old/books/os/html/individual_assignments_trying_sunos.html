1. Trying the SunOS

  1. Faça boot local no micro.

  2. Rode telnet e log numa máquina Unix

  3. Rode o X server no micro. No Unix set a variável DISPLAY para a sua máquina:

    > setenv DISPLAY "IP da sua máquina": 0
> xterm&

    Para descobrir o IP da sua máquina olhe em Start -> Settings -> Control Panel.

    Click em Network , Double Click em TCP/IP e você verá endereço NÃO MODIFIQUE!

    Qual a vantagem de se usar um X server sobre o programa telnet?

  4. Navegue no File System Unix.

  5. Marque os FS que estão sendo montados remotamente na oca e na ceci. Olhe em /etc/fstab.

  6. Identifique um link. Anote-o e explique o que m link.

  7. Verifique os processos que estão rodando ( ps -aux ). Identifique os seus.

  8. Tente localizar e logar em outras máquinas na rede.

  9. Verifique o arquivo de passwords em duas máquinas diferentes (/etc/passwd). Responda: Qual o formato do arquivo /etc/passwd ?

  10. Faça uma descrição do ambiente definido para sua área ( env ).

  11. O que é um álias? Insira um álias diferente (com o nome de um membro do grupo) no seu de ambiente.

  12. Identifique cada um dos arquivos de definição de ambiente (.login e .cshrc). Explique a função de cada um.
