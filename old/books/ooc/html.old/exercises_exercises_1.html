1. Basic Exercises 1

  1. Um ‘signed data type’ possui um número de valores disponíveis ‘não-zero’ positivos e negativos diferentes.

    1. Verdadeiro

    2. Falso

  2. Escolha o identificador válido:

    1. Big01LongStringWithMeaninglessName

    2. $int

    3. bytes

    4. $1

    5. finalist

  3. Quais das seguintes representações são válidas para a entrada do método main() de uma aplicação?

    1. public static void main()

    2. public static void main(String arg[])

    3. public void main(String [] arg)

    4. public static void main(String[] args)

    5. public static int main(String [] arg)

  4. Se todos os três elementos (import, declaração de package and classes) ocorrerem em um arquivo fonte, eles devem aparecer em que ordem?

    1. Imports, declaração de package, classes

    2. Classes, imports, declaração de package

    3. Declaração de Package deve aparecer primeiro; a ordem para imports e definição de class não é significativa

    4. Declaração de package, imports, classes.

    5. Imports devem aparecer antes; a ordem para declaração de package e definição de class não é significante

  5. Considere a seguinte linha de código:

         int  x[] = new int [25];

    Após a execução, qual(is) afirmação(ões) é(são) verdadeiras?

    1. x[24] é 0

    2. x[24] é indefinido

    3. x[25] é 0

    4. x[0] é nulo

    5. x.length é 25

  8. Como se força a ‘garbage collection’ de um objeto?

    1. Garbage collection não pode ser forced

    2. Call System.gc()

    3. Call System.gc(), passando na referência para o objeto a ser ‘garbage-collected’

    4. Call Runtime.gc()

    5. Setar todas referencias para o objeto para novos valores (Ex: null)

    3. 0 a 232 - 1

    4. -215 a 215 – 1

    5. -231 a 231 – 1

  10. Qual é a faixa de valores que pode ser designada para uma variável do tipo byte?

    3. 0 a 216 - 1

    4. –27 a 27 – 1

    5. –215 a 215 – 1
