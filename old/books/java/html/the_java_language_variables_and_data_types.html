1. Variables and Data Types

1.1. Variable Types

Java is a strongly typed language. It tests type information even at runtime.

Caution

You can not force a type upon a variable as you do in C. The following program will result in an error:

String var1= new String("foo");
URL var2= (URL) var1;

1.3. Reference Data Types

Reference types are called such because the value of a reference variable is a reference (a pointer in other terminology) to the actual value or set of values represented by the variable.

Figure 2.1. Pointers in Java

1.3.1. Evaluation

When used in a statement or expression, the name args evaluates to the address of the memory location where the array lives. This is in contrast to the name of a primitive variable, the count variable, which evaluates to the variable's actual value.

Caution

The following C types are not supported by Java:

Pointer, Struct, Union

1.4. Variable Names

In Java, a variable name:

  • must be a legal Java identifier comprised of a series of Unicode characters. This is important so that programmers can write code that is meaningful in their native languages;

  • must not be the same as a keyword or a boolean literal (true or false);

  • must not have the same name as another variable whose declaration appears in the same scope.

1.5. Scope

A variable's scope is the block of code within which the variable is accessible. Also, a variable's scope determines when the variable is created and destroyed. Scope places a variable into one of these four categories:

  • member variable;

  • local variable

  • method parameter

  • exception handler parameter

1.6. Variable Initialization

Local variables and member variables can be initialized when they are declared:

int count = 0;

1.7. Reference type Initialization

A reference type when is created its pointer is initialized to NULL.

All of them must be initialized before use:

String name= new String("Java");

or

String name;
name= new String("Java");
Caution

Objects in Java have to be always explicit created using the command new.
